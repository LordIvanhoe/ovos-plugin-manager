# OVOS plugin manager

OPM can be used to search, install, load and create plugins for the OpenVoiceOS ecosystem!

STATUS: alpha release 1

## Install

```bash
pip install ovos-plugin-manager==0.0.1a1
```

## Usage

see [./examples](examples) folder for basic usage
